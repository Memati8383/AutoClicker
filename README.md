# AutoClicker

Android için floating overlay üzerinden çalışan basit ve kullanışlı bir otomatik tıklama uygulaması.

## Özellikler

- Floating overlay ile kolay kontrol
- Ayarlanabilir tıklama hızı ve aralığı
- Tek / çift tıklama desteği
- Accessibility Service tabanlı çalışır
- Material 3 tasarım

## Kurulum

APK'yı [Releases](https://github.com/Memati8383/AutoClicker/releases) sayfasından indirip telefonuna yükleyebilirsin.

## Kullanım

1. Uygulamayı aç
2. Accessibility Service iznini ver
3. Floating overlay butonuna bas
4. Tıklama ayarlarını yap ve başlat

## Gereksinimler

- Android 7.0+ (API 24+)
- Accessibility Service izni
